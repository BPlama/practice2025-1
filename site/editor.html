Описание проекта

Редактор текста на Python — это часть проектной практики, направленная на изучение языка Python и библиотеки Tkinter для создания GUI-приложений.

Возможности редактора:

  • Открытие и сохранение файлов.
  • Редактирование текста в окне.
  • Поиск слов с подсветкой.
  • Горячие клавиши (Ctrl+O, Ctrl+S, Ctrl+F).

Этапы разработки

  1. Создание основного окна приложения.
  2. Добавление меню с пунктами "Открыть", "Сохранить", "Выход".
  3. Вставка текстового поля для редактирования.
  4. Реализация поиска текста по слову с подсветкой найденного.
  5. Добавление панели поиска и кнопки "Найти".

Пример кода

Создание окна и меню:

root = tk.Tk()
root.title("Редактор текста")
root.geometry("800x600")

Добавление текстового поля:

text_area = tk.Text(root, wrap='word')
text_area.pack(expand=1, fill='both')

Поиск слова в тексте:

def find_text():
    text_area.tag_remove('highlight', '1.0', tk.END)
    search_word = search_entry.get()
    if search_word:
        start_pos = '1.0'
        while True:
            start_pos = text_area.search(search_word, start_pos, stopindex=tk.END)
            if not start_pos:
                break
            end_pos = f"{start_pos}+{len(search_word)}c"
            text_area.tag_add('highlight', start_pos, end_pos)
            text_area.tag_config('highlight', background='lightblue')
            start_pos = end_pos
