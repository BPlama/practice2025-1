Цель проекта

В рамках данного семестра наша команда работает над автоматизацией процесса анкетирования в образовательной среде. Сейчас этот процесс в СДО организован вручную: составление анкет, отбор вопросов и последующий анализ ответов занимают значительное количество времени и требуют ручной работы. Мы ставим перед собой задачу упростить этот процесс, сделав его более эффективным и удобным.

Основные задачи

  • Проанализировать существующие подходы к анкетированию и собрать требования.
  • Разработать структуру базы данных для хранения информации о группах, преподавателях и вопросах.
  • Составить XML-структуру анкет и реализовать генератор анкет с возможностью добавления и изменения вопросов.
  • Создать инструменты для автоматического анализа полученных ответов.
  • Подготовить тестовую и пользовательскую документацию.
  • В дальнейшем — реализовать веб-интерфейс для работы с анкетами.

Используемые технологии

На текущем этапе разработки используются:

  • XML — для хранения анкет, вопросов, полученных ответов после тестирования.
  • Python — для обработки данных, генерации XML-файлов и написания вспомогательных скриптов.
  • Java — для создания модуля анализа и возможного взаимодействия с базой данных.

HTML и CSS планируются к использованию на следующем этапе для создания веб-интерфейса.

Состав команды

Работа распределена между четырьмя подгруппами:

  • Дизайн интерфейса — отвечает за макеты экранов и структуру взаимодействия.
  • Архитектура базы данных — разрабатывает структуру хранения и связи данных.
  • Тестирование — составляет план тестирования и проверяет работоспособность модулей.
  • Создание анкет — занимается генерацией анкет и проведением тестирования.
